# 🛡️ TryHackMe Write-ups

Welcome to my TryHackMe write-ups repository!
This repository reflects my hands-on journey in cybersecurity, where I actively solve labs and document my approach to strengthen my practical skills.

---

## 📌 About This Repository

This repository contains detailed write-ups of various TryHackMe rooms and machines.
Each write-up is carefully structured to showcase not just the solution, but the **thought process and methodology** behind it.

---

## 🚀 What I Focus On

* 🔍 **Reconnaissance** – Identifying open ports and services
* 🕵️ **Enumeration** – Discovering hidden paths, files, and vulnerabilities
* 💥 **Exploitation** – Gaining initial access using identified weaknesses
* 🧗 **Privilege Escalation** – Achieving higher-level access
* 🧠 **Understanding Concepts** – Learning *why* each step works

---

## 🧠 Learning Approach

* I prioritize **understanding over memorization**
* I document both **successful and failed attempts**
* I continuously refine my approach with each machine
* I focus on building **real-world problem-solving skills**
